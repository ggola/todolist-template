template for todolist web app

Deployed in Heroku
https://limitless-chamber-91803.herokuapp.com/
with data persisted in mongoDB Atlas

FEATURES
- add todos for current day list
- add new list by adding /new-list-name to the url
- see all todos in home page
- data persisted with mongoDB (you need to install mongoDB and run it locally)
